- jQuery
- Bootstrap

# Цикл

```
manifest

content_script -> backgrount -> popup
                             -> options
```

- `content_script`: запуск на вкладке
- `backgrount`: запуск с браузером
- `popup`: запуск при открытии всплывающего
- `options`: запуск при открытии страницы опций
